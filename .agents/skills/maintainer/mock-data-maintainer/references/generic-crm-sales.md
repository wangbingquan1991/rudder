# Generic CRM And Sales Data

Use this when the user needs realistic CRM, sales, account, or support data.

## Useful Entities

- accounts
- contacts
- opportunities
- pipeline stages
- activities
- support tickets
- renewal notes
- customer-success tasks

## Pipeline Stages

- prospecting
- discovery
- technical evaluation
- security review
- procurement
- closed won
- closed lost

## Scenario Ideas

### Enterprise Pipeline Review

Show 8-12 opportunities across stages, including one blocked in security
review, one large expansion, and one churn-risk renewal.

### Customer Support Triage

Show support tickets from multiple accounts with severity, owner, SLA, and
linked opportunity or renewal risk.

### Founder Sales Dashboard

Show a small startup pipeline with founder-led sales, high variance deal sizes,
and a few specific next steps.

## Data Quality

- Give each account a believable industry and size.
- Include next action and owner for open opportunities.
- Include at least one stale opportunity.
- Include one closed-lost reason so charts are not happy-path only.
